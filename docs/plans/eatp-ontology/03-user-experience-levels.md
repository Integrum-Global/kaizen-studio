## Implementation Notes

### Role Detection

```typescript
// Determine user experience level based on capabilities
function getUserExperienceLevel(user: User): ExperienceLevel {
  const capabilities = user.effectiveCapabilities;

  if (capabilities.includes('establish_trust') ||
      capabilities.includes('manage_value_chains')) {
    return ExperienceLevel.VALUE_CHAIN_OWNER;
  }

  if (capabilities.includes('delegate_process') ||
      capabilities.includes('manage_processes')) {
    return ExperienceLevel.PROCESS_OWNER;
  }

  return ExperienceLevel.TASK_PERFORMER;
}
```

### Progressive Disclosure

- Level 1: Minimal UI, task-focused
- Level 2: Expanded UI with orchestration visibility
- Level 3: Full UI with governance features

This is implemented via:
1. Conditional sidebar items
2. Feature flags based on capabilities
3. Route guards for protected sections
